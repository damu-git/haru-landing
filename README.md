# 하루 (Haru) 랜딩 페이지

> AI 친구와 대화하고, 진짜 인연을 만나다

하루(Haru)는 AI 친구와의 일상 대화를 통해 정서 기반으로 사람을 연결해주는 서비스입니다. 이 레포지토리는 하루 서비스의 랜딩 페이지입니다.

## 서비스 소개

- **AI 친구 채팅**: 언제든 대화할 수 있는 AI 친구
- **대화 기반 일기**: 하루 대화를 자동으로 일기로 정리
- **주간 분석 리포트**: 사고 패턴과 감정 변화 분석
- **정서 기반 매칭**: 외모가 아닌 마음으로 연결

## 기술 스택

- **Vite** - 빌드 도구
- **TypeScript** - 타입 안전성
- **Tailwind CSS** - 스타일링
- **Vercel** - 배포

## 개발

```bash
# 의존성 설치
pnpm install

# 개발 서버 실행
pnpm dev

# 프로덕션 빌드
pnpm build
```
