## 2. 우리가 해결하려는 문제

### 기존 데이팅앱의 구조적 한계

현재 데이팅앱 시장은 **외모 기반 스와이프 방식**이 주류입니다. 그러나 이 방식에는 근본적인 문제가 있습니다.

| 문제점 | 설명 |
|--------|------|
| **외모 줄세우기** | 사진 한 장으로 판단당하는 부정적 경험 |
| **낙인 효과** | "데이팅앱 사용 = 부끄러움"이라는 심리적 부담 |
| **높은 이탈률** | 매칭되어도 대화가 이어지지 않는 문제 |
| **부자연스러움** | 처음부터 "연애 목적"으로 만나는 인위적 만남 |

### 놓치고 있는 시장

> **데이팅앱을 "절대 안 쓸" 사람들이 상당수 존재합니다.**

이들은 연애를 포기한 게 아닙니다. 기존 방식이 맞지 않는 것입니다.

- 외모 경쟁이 불편한 사람들
- 자연스러운 만남을 원하지만 기회가 없는 사람들
- 온라인에서 먼저 친해지는 게 편한 내성적인 사람들
- 이성 친구 자체가 적어서 기회가 없는 사람들

### 현실의 연애는 어떻게 시작되나요?

대부분의 연애는 **"처음부터 연애 목적"이 아닌 자연스러운 관계**에서 시작됩니다.

```
직장동료 → 스몰토크 → 코드가 맞음 → 더 많은 시간 → 자연스럽게 연인
```

우리는 이 자연스러운 흐름을 앱에서 구현합니다.

```
AI 친구 → 일상 대화 → 정서 파악 → 잘 맞는 사람 소개 → 친구로 시작 → 자연스럽게 발전
```

## 3. 하루의 솔루션

### 핵심 컨셉

> **"데이팅앱"이 아닌 "AI 친구앱"**

사용자는 AI 친구와 일상적인 대화를 나눕니다. AI는 이를 통해 사용자의 정서적 특성을 파악합니다. 충분한 이해가 쌓이면 AI가 자연스럽게 제안합니다.

> *"너랑 잘 맞을 것 같은 친구가 있는데, 얘기해볼래?"*

### 기존 서비스와 차별점

**vs 기존 데이팅앱**

| 구분 | 기존 데이팅앱 | 하루 |
|------|--------------|------|
| **매칭 기준** | 외모, 학벌, 직업 | 정서, 성격, 취향 |
| **데이터 수집** | 프로필 직접 작성 | AI 대화 통해 자연스럽게 축적 |
| **만남 목적** | 처음부터 연애 | 친구로 시작 |
| **판단 방식** | 사진 보고 즉시 스와이프 | 충분한 대화 후 AI가 추천 |
| **심리적 부담** | 높음 | 낮음 (AI 친구앱) |
| **거절 경험** | 직접적 (스와이프 아웃) | 간접적 (AI가 중간 버퍼) |

**vs 기존 AI 챗봇**

| 구분 | 기존 AI 챗봇 | 하루 |
|------|-------------|------|
| **대화 방식** | 질문 → 즉시 답변 | 메시지 → 나중에 답장 올 수도 |
| **느낌** | 도구 | 친구 |
| **상호작용** | 사용자가 시작 | AI가 먼저 연락하기도 함 |
| **UI** | AI 챗봇 UI | 메신저 UI (카톡처럼) |
| **목적** | AI와의 관계 | 실제 사람과의 연결 |

## 4. 핵심 가치 제안

### 사용자가 얻는 가치

#### 1. 언제든 대화 가능한 친구
- 시간과 장소에 구애받지 않고 대화 가능
- 판단 없이 들어주는 존재
- 비밀 보장이 확실한 상대

#### 2. 대화를 통한 자기 이해
- 일상 대화가 일기로 자동 정리
- 주간 리포트로 자신의 패턴 발견
- 무의식적 편향, 반복되는 맹점 인식

#### 3. 진짜 친구를 만나는 새로운 방법
- 외모가 아닌 정서적 궁합 기반 매칭
- 사용자가 찾아 헤매지 않아도 됨
- 거절의 고통이 분산됨 (AI가 버퍼)

### 감정적 편익

- **안전함**: 판단받지 않고 털어놓을 수 있음
- **이해받음**: 나를 알아가는 AI가 곁에 있음
- **희망**: 나와 맞는 사람을 만날 수 있다는 기대감
- **자연스러움**: 억지스럽지 않은 만남

## 5. 타겟 사용자

### 주요 페르소나

#### 페르소나 1: 연애 경험이 적은 20대 후반~30대
- 연애를 안 해본 건 아니지만 많지 않음
- 데이팅앱의 "외모 경쟁"이 불편함
- 자연스러운 만남을 원하지만 기회가 없음

> *"사진 보고 스와이프하는 게 너무 싫어요. 그냥 자연스럽게 알아가고 싶은데..."*

#### 페르소나 2: 내성적이고 사회적 활동이 적은 사람
- 새로운 사람 만나는 게 에너지 소모가 큼
- 온라인에서 먼저 친해지는 게 편함
- 깊은 대화를 좋아하지만 상대가 없음

> *"처음 만난 사람이랑 어색하게 대화하는 게 너무 힘들어요. 어느 정도 알고 만나면 좋겠어요."*

#### 페르소나 3: 이성 친구 자체가 적은 사람
- 동성 친구는 있지만 이성과 대화 기회가 적음
- 연애까지는 아니어도 이성 친구는 갖고 싶음
- "친구 소개"는 부담 없이 받아들일 수 있음

> *"여자(남자) 친구가 거의 없어요. 연애 아니어도 그냥 편하게 얘기할 수 있는 친구가 있으면..."*

### 타겟이 아닌 사용자

- 빠른 만남/캐주얼 데이팅을 원하는 사람
- 외모 기반 매칭에 만족하는 사람
- 이미 활발히 데이팅앱을 사용 중인 사람

## 6. 주요 기능

### AI 친구 채팅

**진짜 친구처럼 대화하는 AI**

- 메신저 UI (카카오톡처럼 자연스러운 경험)
- 일상 공유, 고민 상담, 관심사 대화
- AI가 먼저 연락하기도 함 (푸시 알림)
- 항상 즉답하지 않음 (진짜 친구처럼)

> AI가 먼저 연락하는 경험: *"야 요즘 어떻게 지내? 갑자기 생각나서 연락해봤어 ㅎㅎ"*

### 대화 기반 일기

**하루 대화를 자동으로 일기로 정리**

- 매일 자정에 AI가 대화를 요약
- 감정 변화, 주요 이야기, 인사이트 포함
- 대화가 휘발되지 않고 기록으로 남음
- 감정 패턴을 시각화하여 자기 이해

### 주간 분석 리포트

**일주일을 돌아보며 자신을 깊이 이해**

- 사고 패턴, 의사 결정 방식 분석
- 감정 변화 추이, 트리거 요인 파악
- 무의식적 편향, 반복되는 맹점 인식
- 구체적인 개선 조언 제공

### 정서 기반 매칭 (유료, 추후 출시)

**외모가 아닌, 진짜 나와 맞는 사람**

- AI가 축적된 데이터 기반으로 추천
- "데이트"가 아닌 "친구" 컨셉으로 부담 낮춤
- 랜덤 타이밍에 자연스럽게 제안
- 매칭 전/후 AI가 조언 (관계 코칭)

## 9. 기술 하이라이트

### 자연스러운 AI 대화

- **메신저형 UI**: 카카오톡처럼 자연스러운 대화 경험
- **응답 딜레이**: 진짜 친구처럼 바로 답장 안 할 수도 있음
- **AI 먼저 연락**: 사용자가 안 와도 AI가 먼저 메시지 (푸시)
- **장기 기억**: 과거 대화를 기억하고 맥락 유지

### 정서 분석 & 프로필 구축

- **대화 기반 프로필**: 설문 없이 자연스럽게 데이터 축적
- **정서 패턴 분석**: 성격, 취향, 가치관, 대화 스타일 파악
- **임베딩 기반 기억**: "저번에 얘기한 거" 맥락 파악

### 기술 스택

- Kotlin + Spring Boot + PostgreSQL
- AI: Anthropic Claude (Spring AI)
- 벡터 검색: PGVector
- 푸시: Firebase Cloud Messaging
- 결제: RevenueCat

## 13. FAQ

### Q: 하루는 데이팅앱인가요?

A: 하루는 **AI 친구앱**입니다. AI 친구와 대화하는 것 자체가 메인 가치이고, 매칭은 충분한 이해가 쌓인 후 자연스럽게 제안되는 추가 기능입니다. 매칭 기능 없이도 충분히 가치 있게 사용할 수 있어요.

### Q: AI와 대화하면 뭐가 좋은가요?

A: 시간과 장소에 구애받지 않고 언제든 대화할 수 있고, 판단 없이 들어주는 친구가 생겨요. 대화가 일기로 정리되어 자신을 객관적으로 돌아볼 수도 있고요.

### Q: 매칭은 어떻게 이루어지나요?

A: AI가 대화를 통해 사용자의 성격, 취향, 가치관을 파악해요. 그리고 비슷하거나 보완적인 특성을 가진 다른 사용자를 찾아 자연스럽게 소개해줍니다. 설문으로 프로필 만드는 게 아니라, 실제 대화에서 드러나는 진짜 모습을 기반으로 해요.

### Q: 무료로 뭘 할 수 있나요?

A: AI 친구와의 대화, 일기 자동 생성, 주간 분석 리포트 모두 무료예요. 매칭 기능만 프리미엄입니다.

### Q: 내 대화 내용은 안전한가요?

A: 대화 내용은 암호화되어 안전하게 저장됩니다. 개인정보처리방침에 따라 관리되며, 원하시면 언제든 데이터 삭제를 요청하실 수 있어요.
